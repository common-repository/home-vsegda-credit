== Description ==

Хоум Кредит Банк "Всег.Да Кредит" - плагин выводит кнопку "Всег.Да Кредит" от Хоум Кредит Банк на странице корзины WooCommerce.

[Описание и подключение услуги на сайте Всег.Да](https://vsegda-da.com/)

== Frequently Asked Questions ==

= Какие существуют ограничения? =

Дополнительные ограничения по оработе с системой вы можете найти в инструкции по установке виджета «Всег.Да Кредит».

== Installation ==

Установите плагин через панель администратора WordPress.

1. Для работы плагина **Home Vsegda Credit** необходимо наличие установленного и активированного плагина **WooCommerce**.
2. Настройки плагина находятся в меню: `WooCommerce -> Настройки -> Home Vsegda Credit`.
